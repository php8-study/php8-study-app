# Contributing to PHP8 Study

プロジェクトへの貢献を検討していただきありがとうございます！

## 開発環境のセットアップ
1. リポジトリをクローンします
2. `bin/setup` を実行します
3. `bin/dev` でサーバーを起動します

## 修正の提案方法
1. このリポジトリをフォークします
2. ブランチを作成します
3. 変更をコミットし、プッシュします
4. プルリクエストを作成してください

## 注意事項
- 既存のテストがパスすることを確認してください。
- 大規模な変更の場合は、事前に Issue で議論することをお勧めします。
